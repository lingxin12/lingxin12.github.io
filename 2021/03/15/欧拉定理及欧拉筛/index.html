欧拉筛

首先先回顾一下常用的埃氏筛法

void create_prim(int n)
{
    int m = sqrt(n+0.5);
    for(int i=2;i<=m;i++) if(!vis[i])
        for(int j=i*2;j<=n;j+=i) vis[j] = 1;
    for(int i=2;i<=n;i++) prim.push_back(i);
}

该算法的复杂度为O(nlognlogn)，明显已经能够满足我们的需求了，但是当n的数值非常大的时候，仍然非常耗费时间，于是就有了一种线性筛法：欧拉筛。

欧拉筛的原理是：以该数的最小质因子为基础，保证每个数只被筛一遍。

比如$60 = 2^{2} \times 3 \times 5$，而60只能用$2 \times 30$筛去，而不能用$3 \times 20$筛去。

可以在埃氏筛法的基础上写出欧拉筛：

void _create_prim(int n)
{
    for(int i=2;i<=n;i++) {
        if(!vis[i]) prim.pb(i);
        for(int j=0;j<prim.size()&&i*prim[j]<=n;j++) {
            vis[i*prim[j]] = 1;
            if(i%prim[j]==0) break;
        }
    }
}

复杂度为O(n)，其中最核心的一条语句：

if(i%prim[j]==0) break;

该语句保证了我们每个数只被筛一遍，为什么呢？

简单证明一下，如果遇到i%prim[j]==0还不break，那么下一个要筛的数为s = i*prim[j+1]，由于i%prim[j]=0，所以s = k*prim[j]*prim[j+1]，明显prim[j]<prim[j+1]，那么prim[j+1]就不是最小质因子了，那么就会重复筛掉。

（其实这里也能主观上理解一下，prim[j]前面的质数都不是i的因子，一到prim[j]这里prim[j]就变成了i的质因子，那么就能够说明prim[j]作为i*prim[j]的最小质因子将该数筛掉）

欧拉函数

定义：

$\phi(x)$：表示[1,x]中有几个与x互质的数。

通式：

$$ \phi(x) = x \times \prod_{i=1}^{n}(1-\frac{1}{p_i}) $$

性质：

性质1：如果a是素数且n不是a的倍数，有$ \phi(n \times a) = \phi(n) \times (a - 1) $

性质2：如果a是素数且n是a的倍数，有$ \phi(n \times a) = \phi(n) \times a $

性质3：如果n是素数，有$ \phi(n) = n-1 $

性质4：如果n与m互质，那么有$ \phi(n \times m) = \phi(n) \times \phi(m) $

性质5：如果n为奇数，那么$ \phi(2 \times n) = \phi(n) $

性质6：如果n>2，那么$ \phi(n) $为偶数

性质7：如果p是素数，那么$ \phi(p^q) = p^q - p^{q-1} = p^{q-1} \times (p-1) $

求和式：

  1. $ \displaystyle \sum_{d|n}\phi(d) = n $
  2. $ \displaystyle \sum_{i=1}^n[(n,i) = 1] = \phi(n) $
  3. $ \displaystyle \sum_{i=1}^ni[(n,i) = 1] = \lceil \frac{\phi(n) \times n}{2} \rceil $，由于n>2时欧拉函数为偶数，所以只有n=1时为奇数，故也能写成$ \frac{\phi(n) \times n + [n = 1]}{2} $

求欧拉函数值：

根据同时即可用代码在O(sqrt(n))的时间内求出欧拉函数值

int Phi(int n)
{
    int res = n, a = n;
    for(int i=2;i*i<=a;i++) {
        if(a%i == 0) {
            res=res/i*(i-1); //&#x9632;&#x6EA2;&#x51FA;
            while(a%i == 0) a/=i;
        }
    }
    if(a>1) res=res/a*(a-1); //&#x5269;&#x4E0B;&#x7684;&#x9664;&#x4E0D;&#x5B8C;&#x7684;a&#x5FC5;&#x4E3A;&#x8D28;&#x6570;
    return res;
}

可以利用性质1，2来模仿欧拉筛在O(n)的时间内递推出n以内的欧拉函数值，同时筛出质数

void Get_Phi()
{
    phi[1] = 1;
    for(int i=2;i<=maxn;i++) {
        if(!vis[i]) {
            prim.pb(i);
            phi[i] = i-1;
        }
        for(int j=0;j<prim.size()&&i*prim[j]<=maxn;j++) {
            vis[j] = 1;
            if(i%prim[j] == 0) {
                phi[i*prim[j]] = phi[i] * prim[j];  //&#x6027;&#x8D28;2
                break;
            }
            else phi[i*prim[j]] = phi[i] * phi[prim[j]];  //&#x6027;&#x8D28;1
        }
    }
}

如果我们要求的欧拉数已经超过了n，可以利用性质4将n拆成多个不同的质因数的欧拉函数值，再利用性质3，7求解

int _Phi(int n)
{
    int res = 1;
    for(int i=0;i<prim.size();i++) {
        if(n < maxn) return res*phi[n];
        if(n%prim[i] == 0) while(1) {
            n/=prim[i];
            if(n%prim[i] == 0) res*=prim[i];
            else {
                res*=prim[i]-1;
                break;
            }
        }
        if(n == 1) return res;
        if(prim[i]>sqrt(n)) return res*(n-1); //&#x8FD9;&#x91CC;&#x8BF4;&#x660E;n&#x5DF2;&#x7ECF;&#x4E3A;&#x8D28;&#x6570;&#x4E86;
    }
}

欧拉定理

$$ a^{\phi(m)} \equiv 1 (mod m) ((a,m) = 1) $$

欧拉降幂

当a与m互质时：

$ a^q \equiv a^r (mod m) $

其中$ q = k\phi(m) + r $

通式（不论是否互质）：

当$ q \geq \phi(m) $时：

$ a^q \equiv a^{r + \phi(m)} (mod m) $

其中$ q = k\phi(m) + r $
