威尔逊定理

内容

当且仅当p为质数时，有$ (p-1)! \equiv -1 (mod \ p) $

证明

（1）必要性：

若$ p|(p-1)!+1 $且p不为质数，则有$ p=ab,\ \ a|p|(p-1)!+1 $，由于$ a|(p-1)!,\ \ a \nmid 1,\ \ a \nmid (n-1)!+1 $，矛盾。

（2）充分性：

p为质数，设$ M = ${$2,3,…,p-2$}$ , \ \ N = ${$1,2,3,..,p-1$}$ $，

对$ \forall a \in M,\ \ (a,p) = 1,\ \ $，取集合$ S = a·N = ${$a,2a,…,a(p-1)$}$ $

对$ \forall x \in S,\ \ (x,p) = 1 $

则有$ S \equiv N (mod \ p) $（具体见附证一）

上式说明对于任意一个a属于M，a与集合N每一个数相乘都与集合N中一个数同余，那么肯定存在a与集合N中一个数相乘模p余数为1。

即$ \forall a \in M, \ \ \ \exists x \in N, \ \ \ ax \equiv 1 (mod \ p) $。

容易证明当x=1或a或(p-1)时与已知矛盾。

上述说明了可以在集合M中找到任意两个不同的数，它们之积在mod p的情况下与1同余。

则$ (p-1)! \equiv 2 \times 3 \times … \times (p-1) \equiv (p-1) \equiv -1 \ \ (mod \ p) $

附证一

证明：

在(mod p)的情况下，有

$$ a \equiv a_1 \\ 2a \equiv a_2 \\ … \\ (p-1)a \equiv a_{p-1}$$

其中$ a_1,a_2,…,a_{p-1} $各不相同。

假设$ a_i = a_j ,\ \ (i < j) $，有$ ia \equiv a_i ,\ \ \ ja \equiv a_j \Rightarrow \ \ p|ia-a_i \ \ p|ja-a_j \Rightarrow \ \ p|ja-ia=(j-i)a$。

但是p为质数，且$ j-i < p,\ \ a < p $，则$ (p,j-i)=1,\ \ (p,a) = 1 $，与上述$ p|(j-i)a $矛盾。

应用

求$ \displaystyle \sum_{k=1}^{n} [\frac{(3k+6)!+1}{3k+7} - [\frac{(3k+6)!}{3k+7}]] $，$[]$表示向下取整。

（1）当$ 3k+7 $为质数时，根据威尔逊定理，$ \frac{(3k+6)!+1}{3k+7} $为整数，设为K。

$ [\frac{(3k+6)!}{3k+7}] = [\frac{(3k+6)!+1-1}{3k+7}] = [\frac{(3k+6)!+1}{3k+7}-\frac{1}{3k+7}] = [K-\frac{1}{3k+7}] = K-1 $。

此时原式 $ = [K-(K-1)] = 1 $。

（2）当$ 3k+7 $不为质数时，$ 3k+7 = ab，a \leq 3k+6, \ \ b \leq 3k+6 $，故$ ab=3k+7 | (3k+6)! $。

则原式 $ = [\frac{(3k+6)!}{3k+7}+\frac{1}{3k+7} - \frac{(3k+6)!}{3k+7}] = [\frac{1}{3k+7}] = 0 $。

若$ 3k+7 = a^2 $（a为质数，若a不为质数，一定能够拆成ab，$a \neq b$），当$ a \leq 7 $时，不存在该式。

当$ a > 7 $时，$ 3k+7 = a^2 > 2a \ \Rightarrow \ \ 3k+6 \geq 2a $，则$ (3k+6)! $有$a$与$2a$的因子，因此$ [\frac{(3k+6)!}{3k+7}] = \frac{(3k+6)!}{3k+7} $
