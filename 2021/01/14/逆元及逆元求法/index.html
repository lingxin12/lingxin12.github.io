    逆元：

    ax=1(mod p)中x称为a关于p的逆元。

    主要用于除法取模。

    若bx=1(mod p)，那么 (a/b) mod p = ((a mod p) * x ) mod p。

    证明：

    a/b = c(mod p) , bx=1(mod p) => (a/b)*(bx) = c*1(mod p) => a*x = c (mod p)

    逆元的三种求法：

    1.当p为质数的时候，用费马小定理

    费马小定理：若p为质数，且a不为p的倍数，则a^p=a(mod p)。

    将上式转换一下：a*(a^(p-2)) = 1 (mod p) ，显然a^(p-2)为逆元。

    用取模快速幂求。

    int fastpow(int a,int b,int p)
{
    int res = 1;
    while(b) {
        if(b&1) res = (res*a)%p;
        b >>= 1;
        a = (a*a)%p;
    }
    return res;
}
int inv(int a,int p) { return fastpow(a,p-2,p); } //a&#x5173;&#x4E8E;p&#x7684;&#x9006;&#x5143;

    2.当p不为质数的时候，可以用扩欧

    对于ax=1(mod p)，有p|ax-1，那么存在y使得py=ax-1，即ax+p(-y)=1。

    用扩欧求得的x即为a关于p的逆元。

    对于ax+by=1，两边同时对b取模有

    (ax mod b) + (by mod b) = (1 mod b) 那么 ax mod b = 1 即 ax=1(mod b)

    这里的x就是a关于b的逆元，如对a取模有同样道理。

    那么很明显的发现：

    对于ax+by=1，x是a关于b的逆元，y是b关于a的逆元。

    代码如下：

    void ex_gcd(int a, int b, int& d, int& x, int& y)  //&#x6269;&#x5C55;&#x6B27;&#x51E0;&#x91CC;&#x5F97;
{ 
    if(!b) { 
        d = a; x = 1; y = 0; 
    } else { 
        ex_gcd(b, a%b, d, y, x); y -= x*(a/b); 
    } 
}
int inv(int a,int p)  //&#x5982;&#x679C;&#x4E0D;&#x5B58;&#x5728;&#x8FD4;&#x56DE;-1
{
    int d,x,y;
    ex_gcd(a,p,d,x,y);
    return d == 1 ? (x%p + p) % p : -1;  //x&#x53EF;&#x80FD;&#x4E3A;&#x8D1F;&#x6570;
}

    3.当p为质数时，可以在O(n)的时间内求多个数的逆元

    先说一下结论：

    当p为质数时有 inv(a) = (p - p / a) * inv(p % a) % p 。 （其中inv(a)代表a关于p的逆元）

    当a为1时逆元为1

    证明如下：

    设x = p % a , y = p / a

    于是有 x + y * a = p （做带余除法）

    x + y * a = 0 (mod p)

    移项有 x = -y * a (mod p)

    x * inv(a) = (-y) (mod p) （同时乘上a的逆元）

    inv(a) = (p - y) * inv(x) (mod p) （同时乘上x的逆元）

    于是 inv(a) = (p - p / a) * inv(p % a) % p （把x，y代入）

    先看只求一个数的代码：
