背包九讲学习笔记

2021-05-11
算法 > 动态规划-ACM动态规划背包问题

参考博客（内有相关例题）：https://blog.csdn.net/yandaoqiusheng/article/details/84782655/

01背包

方程

f[i][j]为将前i件物品放入容量为j的背包中的最大价值
f[i][j]=max{f[i-1][j],f[i-1][j-v[i]]+w[i]}

空间上的优化

去掉数组的第一维，在更新一个i时，f数组存的其实就是上一轮的结果，因此为了保证j-v[i]是上一轮的结果，需要j从V往v[i]输出。代码如下：

for(int i=1;i<=n;i++)	
    for(int j=V;j>=v[i];j--)
        dp[j] = max(dp[j], dp[j-v[i]]+w[i]);

初始化

如果题目要求背包不必装满的情况下的最大价值，那么dp数组初始化为0，如果要求背包恰好装满时的最大价值，那么dp数组除了dp[0]=0以外初始化为-1。

一个常数上的优化

注意到我们最后要访问的结果是f[V]，想要得出这个答案，那么在上一轮的时候我们只需要知道f[V]和f[V-v[n]]然后比较其大小即可。而相对于第n-1轮来说，只需要知道f[V]，f[V-v[n]]，f[V-v[n]-v[n-1]]然后在相应的轮数比较相应的大小即可。因此在第i轮，只需要循环从V到f[V-sum(v[i+1,i+2,……,n])]即可。求sum用前缀和即可。

for (int i = 1; i <= n; i++) cin >> v[i] >> w[i], s[i] = s[i - 1] + v[i];
for (int i = 1; i <= n; i++) {
    int bound = max(v[i], V - (s[n] - s[i]));
    for (int j = V; j >= bound; j--)
        dp[j] = max(dp[j], dp[j - v[i]] + w[i]);
}

完全背包问题

方程

f[i][j] = max{f[i-1][j], f[i-1][j-k*v[i]]+k*w[i]}} (0<=k*w[i]<=j)

代码

for(int i=1;i<=n;i++)
    for(int j=v[i];j<=V;j++)
        dp[j] = max(dp[j], dp[j-v[i]]+w[i]);

多重背包问题

方程

f[i][j] = max{f[i-1][j], f[i-1][j-k*v[i]]+w[i]} (0<=k<=p[i], 0<=k*v[i]<=j)

转化为01背包

用二进制思想（如1，2，4，8通过选择其中的若干个数，可以组成1到15任何一个数），可以将p[i]分成1,2,4,8,…,$ p[i]-2^{k}+1 $，其中$ p[i]-2^{k}+1 $ > 0。用二进制的思想易证明用前边的数能够组成1到p[i]任何一个数。比如22分成了1，2，4，8，7，这些数全部加起来正好为22。所以可以把第i件物品分成若干件物品，然后就能转化成01背包了。

代码

for(int i=1;i<=n;i++)
{
    int num = min(p[i], V/v[i]);
    for(int k=1;num>0;k<<=1) {
        if(k > num) k = num;
        num-=k;
        for(int j=V;j>=k*v[i];j--)
            dp[j] = max(dp[j], dp[j-k*v[i]]+k*w[i]);
    }
}

void MultiPack(int p, int w, int v) {
    for (int j = 0; j < w; j++) { //j&#x4E3A;w&#x7684;&#x6240;&#x6709;&#x7EC4;
        int head = 1, tail = 0;
        for (int k = j, i = 0; k <= V / 2; k += w, i++) {
            int r = f[k] - i * v;
            while (head <= tail and r >= q[tail].v) tail--;
            q[++tail] = node(i, r);
            while (q[head].id < i - p) head++; //&#x9700;&#x8981;&#x7684;&#x7269;&#x54C1;&#x6570;&#x76EE;
            f[k] = q[head].v + i * v;
        }
    }
}
