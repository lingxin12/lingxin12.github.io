几种常见的博弈

一、前置知识

参考博客：https://blog.csdn.net/lgdblue/article/details/15809893

算法竞赛中博弈论相关题目的特点：

1.两人进行一场游戏，轮流决策，两个人每一步用的都是最优策略。

2.博弈是有限的，即两人无论怎样决策，最终都会决出胜负。

3.公平博弈，即两个进行决策所遵守的规则相同。

理论铺垫：

必胜态（N态）：一定会胜利的状态。

必败态（P态）：一定会失败的状态。

特点：

（我们把一个局面当成一个节点，做一个决策会从一个局面走到另外一个局面，那么就可以把一个决策看成一条有向边，那么博弈的过程实际上就是在一张图上从一个局面走到最终局面的过程）

1.无法进行任何移动为P

2.可以移动到P的局面是N

3.所有移动都会走到N的局面是P

（后两点也就是说：P怎么移动都只能走到N，而N移动可以走到P也可以走到N，但是必有一条路走到P）

性质：

1.若当前状态已经无法进行决策为必败态。

2.一个局面为必胜态的充要条件是进行某种决策后变成必败态。

3.一个局面为必败态的充要条件是无论进行何种决策均会变成必胜态。

取石子游戏算法实现

步骤1:将所有终结位置标记为必败点。

步骤2: 将所有一步操作能进入必败点的位置标记为必胜点。

步骤3:如果从某个点开始的所有一步操作都只能进入必胜点。则将该点标记为必败点。

步骤4: 如果在步骤3未能找到新的必败，则算法终止。否则，返回到步骤2。

二、巴什博弈

1.问题模型：

有一堆石头，一共有n个，两个人轮流从这堆石头取石头，规定最少取一个最多取m个，最后取光者胜。

2.解题思路：

显然当n=0时先手必败，明显发现当1<=n<=m时，先手可必胜。

考虑n=m+1，无论做什么决策，最终n都会在[1,m]这个区间中，所以此时先手必败。

考虑n=m+2，无论做什么决策，最终n在[2,m+1]区间中，也就是说此时有一种决策可以走向必败态，本着求胜的原则，此时先手必胜。

显然n从m+3到m+m+1都为相同情况。

当n=2*(m+1)，决策后n在[m+2,m+m+1]中，只能走向必胜态，所以此时为必败态。

此时已经不难发现了，当(m+1)|n时，后手必胜。

3.结论：

当(m+1)|n时后手必胜。

4.变形：

条件不变，最后取光的人输。

三、尼姆博弈

1.问题模型：

有n堆石子，第i堆有$a_i$个石子，两个人轮流在某一堆石子中取任意多个（最少一个，多者不算），最后取光者胜。

2.解题思路：

显然每堆石子个数都为0时为必败态。

考虑n堆石子个数的异或和为0时：$a_1$^$a_2$^……$a_i$^……^$a_n$ = 0

此时从某堆拿走石子：S = $a_1$^$a_2$^……$a ^{‘} _i$^……^$a_n$

显然 S $\neq$ 0，若不然，根据异或和的可结合律可得$a_i$ = $a ^{‘} _i$，矛盾。

而最终态的异或和为0（必败态），则上面可说明只要异或和不为0，则有办法移动到异或和为0的局面，即当异或和不为0时，为必胜态。

3.结论：

当n堆石子个数异或和为0时，后手必胜。

四、斐波那契博弈

1.问题模型：

有一堆个数为n的石子，两人轮流取石子，满足：

（1）先手不能在第一次取完所有石子。

（2）之后每次可以取的石子数介于1到对手刚取的石子数的两倍之间（包括这两个数）。

（3）约定取走最后一个石子的人为赢家。

2.解题思路：

当n为斐波那契数时，先手必败。

根据齐肯多夫定理：任何正整数可以表示为若干个不连续的斐波那契数之和。如n=83=55+21+5+2时，假如先手取2个，显然后手就不能取5个，所以后手取的那5个中必有剩下，那么再把剩余的石子取走，然后后手又不能取完21个，那么再把21个中剩余的取走，那么最终先手可以取完最后一堆石子，所以n不为斐波那契数时可以按照以上方式先手必胜。

下面证明n为斐波那契数时先手必败。

设n = F(t) = F(t-1)+F(t-2)，先手取y个石子，下面分两种情况：

（1）若y $\geq$ F(t-2)，那么剩下的石子数量为F(t)-y $\leq$ F(t)-F(t-2) = F(t-1) = F(t-2)+F(t-3)<2F(t-2) $\leq$ 2y，所以后手一定能把所有石子取完，此时先手必败。

（2）若y < F(t-2)，设t充分大，

当t为偶数时有F(t) = F(t-1) + F(t-3) + …… + F(3) + F(1)，这里可以看成原来的那堆石子分成了若干堆（每堆都是斐波那契数），若先手取后m堆石子（m$\geq$1），那么就会变成不连续的斐波那契数之和，则按照上述方法后手必胜（这里可以自行证明F(s) < 2[F(s-2)+F(s-4)+……+F(1)]，这个证明告诉我们先手取后m堆石子后，后手可以取完剩下的最后面的那堆石子，然后就变成了上述的情况）。若先手取后m堆石子（m$\geq$1）加上下一堆的，那么最后一堆剩下的石子后手取完，又变成了上述的那种情况，后手必胜。

当t为奇数时，F(t) = F(t-1) + F(t-3) + …… + F(3) + F(1) + F(1)，与偶数情况类似，可自行尝试证明。

综上得证。

3.结论：

当n为斐波那契数时，后手必胜。

4.变形：

胜负条件交换。

五、威佐夫博弈

1.问题模型：

有两堆石子，两人轮流从一堆或同时从两堆中取同样多的石子，每次至少取一个，多者不算，最后取光者获胜。

2.解题思路：

设($a_i$,$b_i$)（$a_i$ $\leq$ $b_i$，i=0,1,2,……,n）表示两堆石子的数量并成为局势，如果局势为(0,0)这是败态，这种局势称之为奇异局势。

我们可以知道前几个奇异局势： (0,0)、(1,2)、(3,5)、(4,7)、(6,10)、(8,13)、(9,15)、(11,18)、(12,20)。（可以发现奇异局势的差是依次递增的）

这里直接给出公式（证明见百度百科），$a_k$=$[\frac{k(1+\sqrt{5})}{2}]$，$b_k$=$a_k + k$（k=0,1,2,……）。（同时$a_k$是未在前面出现过的最小自然数）

上述公式的性质：

（1）任何自然数都包含在一个且仅有一个奇异局势中。

同时$a_k$是未在前面出现过的最小自然数，所以有$a_k > a_k - 1$，而

$$b_k = a_k + k > a_{k-1} + k - 1 = b_k - 1 > a_{k-1}$$

所以$a_k$与$b_k$都不会重复出现。

（2） 任意操作都可将奇异局势变为非奇异局势。

若只改变奇异局势($a_k$,$b_k$)的某一个分量，那么另一个分量不可能在其他奇异局势中，所以必然是非奇异局势。如果使($a_k$,$b_k$)的两个分量同时减少，则由于其差不变，且不可能是其他奇异局势的差，因此也是非奇异局势。（奇异局势的差是依次递增的）

（3） 采用适当的方法，可以将非奇异局势变为奇异局势。

假设面对的局势是(a,b)，若b = a，则同时从两堆中取走 a 个物体，就变为了奇异局势(0,0)。

如果$a = a_k$，$b > b_k$，那么取走$b - b_k$个物体，即变为奇异局势$(a_k,b_k)$。

如果$a = a_k$，$b < b_k$，则同时从两堆中拿走$a_k-a_{b-a_k}$个物体,变为奇异局势($a_b-a_k$,$a_{b-a_k}+b-a_k$)。

如果$a > a_k$，$b = a_k + k$，则从第一堆中拿走多余的数量$a - a_k$即可。

如果$a < a_k$，$b = a_k + k$，分两种情况，第一种，$a = a_j$（j < k），从第二堆里面拿走$b – b_j$即可，变成($a_j$,$b_j$) ，第二种，$a = bj$（j < k），从第二堆里面拿走$b – a_j$即可，变成($a_j$,$b_j$)。

如果$a \neq a_k$，$b \neq b_k$，那么同上分两种情况即可。

3.结论：

$a_k$=$[\frac{k(1+\sqrt{5})}{2}]$，$b_k$=$a_k + k$（k=0,1,2,……）时后手必胜。

4.变形：

胜负条件交换。
