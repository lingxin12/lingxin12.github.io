1002 Graph Theory Class

题意：

对一个有n个顶点的完全无向图，顶点编号从1到n，对于第i个顶点与第j个顶点间的边权为lcm(i+1,j+1)。

求该图的最小生成树的边权和。

分析：

原题很长，实际上问题就是，将2到n+1这n个点，加入到一个连通图中。

1.我们先以最简单的2为根节点，很显然，对于质数，我们想要它产生最小的贡献，最好的方法就是直接与2相连。

2.然后对于合数，连上这棵树所产生的最小贡献即为它本身，比如合数m=x1x2x3······（xi为质数），那么只需要把m节点连到某个xi，就可以产生最小的贡献m。

那么结果就为：2*(S-2)+[n*(n+3)/2 - S]。（S为n+1内所有质数和）。

那么问题就化为了如何求某个范围内的质数和。

1003 Express Mail Taking

题意：

有n个快递柜，编号分别为1到n，两个快递柜之间的距离为1。入口为快递柜1，其中有一个编号为k的快递柜为密码柜，这个密码柜不放快递，但是拿快递的时候必须先要去密码柜输密码，小明为了防止他的快递被偷，所以每次输密码只会拿一个快递。在拿到最后一个快递后回到入口（一共有m个快递）。

分析：

分两种情况去考虑（设密码柜编号为k，设每个快递与密码柜的差值绝对值为mi）。

一、当k左边没有快递的时候，发现无论先取哪个快递都是同一个答案。每个快递与密码柜的距离都贡献了两倍的答案，密码柜和入口的距离贡献了两倍的答案，结果为2*(k-1)+2* (Σ mi)。

二、当k右边有快递的时候，会发现优先留最左边的快递即为最佳答案。假设最坏的情况，每个快递与密码柜的差值的两倍都贡献的答案，那么对于左边最后留下的那个快递（由第一种情况可以知道不需要考虑右边，因为右边留下哪个都对答案不影响），去领的途中和 密码柜的途中 所产生的距离包含在了 从密码柜与入口之间的距离（即领最后一个快递时顺便回入口，且不需要回密码柜，结果为2*(k-1)+2* (Σ mi)-2*mj（mj为最左边快递编号）。

（注意答案为long long）

代码：

    scanf("%lld",&t);
    while(t--)
    {
        scanf("%lld%lld%lld",&n,&m,&k);
        for(int i=1;i<=m;i++) scanf("%lld",&a[i]);
        fla = 0;
        for(int i=1;i<=m;i++) if(a[i]-k < 0) c[i] = k-a[i], fla = max(fla,c[i]); 
                            else c[i] = a[i]-k;
        ll res = 2*(k-1);
        for(int i=1;i<=m;i++) res += 2*c[i];
        if(fla == 0) 
            printf("%lld\n",res); 
        else {
            res -= 2*fla;
            printf("%lld\n",res);
        }
    }
    return 0;
}

1011 3x3 Convolution

题意：

分析：

一、考虑最普通的情况，即为8个0的情况，那么按照题意K矩阵必为只有一个位置为1，其余为0。

1.假设位置在K[1][1]，由样例就可以得出（自己也可以模拟），其实极限就是原矩阵。

2.如果在其他位置的话，通过模拟很容易知道，在一次递推后，会有一些行或列为0，显然在反复递推后，最后的结果就是0。

二、如果K矩阵出现小数，那么必有多个位置出现小数，可以通过一下往下列下去，每个位置都因为多少乘上了小数，最后会越乘越小。

上图是官方给的证明，看起来还挺对，但是看最后一句话就错了，只有8个0出现的时候，极限才是0矩阵。

代码：

#include<bits/stdc++.h>
#define enl puts("")
#define pii pair<int,int>
#define pb push_back
typedef long long ll;
typedef long double ld;
typedef unsigned long long ull;
using namespace std;
int t,n,a[100][100],k[5][5],tot;
int main()
{
    scanf("%d",&t);
    while(t--)
    {
        scanf("%d",&n);
        for(int i=1;i<=n;i++) for(int j=1;j<=n;j++) scanf("%d",&a[i][j]);
        for(int i=1;i<=3;i++) for(int j=1;j<=3;j++) scanf("%d",&k[i][j]);
        tot = 0;
        for(int i=1;i<=3;i++) for(int j=1;j<=3;j++) if(k[i][j] > 0) tot++;
        if(tot == 1) for(int i=1;i<=n;i++) for(int j=1;j<=n;j++) printf("%d%c",a[i][j],j==n?&apos;\n&apos;:&apos; &apos;);
        else for(int i=1;i<=n;i++) for(int j=1;j<=n;j++) printf("0%c",j==n?&apos;\n&apos;:&apos; &apos;);
    }	
    return 0;
}
