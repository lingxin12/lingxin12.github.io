简介

[image: output.png]

该图片是通过games101-homework2光栅化（冯氏着色）出来的图片，放大之后可以明显地发现三角形边缘有着比较严重的锯齿状。

一般要解决问题，达到抗锯齿的效果，大概会有以下几种方法：

  1. 增加屏幕的分辨率（成本较高）
  2. 模糊（滤波）处理后采样（涉及信号处理，不那么熟悉）
  3. SSAA（super sampling anti-aliasing）
  4. MSAA（muti-sample anti-aliasing）
  5. FXAA（fast approximate anti-aliasing）
  6. TAA（tem’poral anti-aliasing）
  7. DLAA（deep learning super sampling）

由于网上的原理已经有很多了，但是基本上没有讲代码实现的，下面着重从代码的角度上实现SSAA、MSAA、FXAA

Aliasing

先看看不使用抗锯齿技术的主要代码：

bool rst::rasterizer::insideTriangle(const float x, const float y, std::array<Vector4f, 3> t) {
    Vector2f t0(t[0](0), t[0](1));
    Vector2f t1(t[1](0), t[1](1));
    Vector2f t2(t[2](0), t[2](1));
    Vector2f p(x, y);

    Vector2f t01(t1(0) - t0(0), t1(1) - t0(1));
    Vector2f t12(t2(0) - t1(0), t2(1) - t1(1));
    Vector2f t20(t0(0) - t2(0), t0(1) - t2(1));
    Vector2f t0p(p(0) - t0(0), p(1) - t0(1));
    Vector2f t1p(p(0) - t1(0), p(1) - t1(1));
    Vector2f t2p(p(0) - t2(0), p(1) - t2(1));

    auto cross = [](Vector2f v1, Vector2f v2) -> float { return v1(0) * v2(1) - v1(1) * v2(0); };

    if(cross(t01, t0p) < 0 || cross(t12, t1p) < 0 || cross(t20, t2p) < 0)
        return false;
    return true;
}

void rst::rasterizer::rasterize_triangle(const Triangle& t) {
    auto v = t.toVector4();
    
    // TODO : Find out the bounding box of current triangle.

    float x_min = std::floor(std::min(v[0](0), std::min(v[1](0), v[2](0)))) - 1,

          y_min = std::floor(std::min(v[0](1), std::min(v[1](1), v[2](1)))) - 1,
          y_max = std::ceil(std::max(v[0](1), std::max(v[1](1), v[2](1)))) + 1;

    // TODO : set the current pixel (use the set_pixel function) to the color of the triangle (use getColor function) if it should be painted.

    for(float pixel_x = x_min; pixel_x < x_max; pixel_x += 1.) {
        for(float pixel_y = y_min; pixel_y < y_max; pixel_y += 1.) {
            float px = pixel_x + 0.5, py = pixel_y + 0.5;
            if(!insideTriangle(px, py, v)) continue;
            
            auto[alpha, beta, gamma] = computeBarycentric2D(px, py, t.v);
            float w_reciprocal = 1.0/(alpha / v[0].w() + beta / v[1].w() + gamma / v[2].w());
            float z_interpolated = alpha * v[0].z() / v[0].w() + beta * v[1].z() / v[1].w() + gamma * v[2].z() / v[2].w();
            z_interpolated *= w_reciprocal;

            int index = get_index(pixel_x, pixel_y);
            if(z_interpolated < depth_buf[index]) {
                depth_buf[index] = z_interpolated;
                // ERROR
                // Because int calculate in set_pixel, you can not use the center pixel, you need correspond the index of frame_buf
                set_pixel(Eigen::Vector3f(pixel_x, pixel_y, 0), t.getColor());
            }
        }
    }
}

上面主要是两个函数，一个是insideTriangle，用于判断某个像素点是否在三角形内，另一个是rasterize_triangle，用于三角形光栅化（结合了z-buffer）。

上面其实犯了一个比较不好的错误，在set_pixel函数中传入的参数不是像素的中心，否则有可能会算不出frame_buf的正确的下标（可以去结合games101-homework2的代码看看）。

void rst::rasterizer::FXAA(const Triangle& t) {
    auto v = t.toVector4();

    float x_min = std::floor(std::min(v[0](0), std::min(v[1](0), v[2](0)))) - 1,
          x_max = std::ceil(std::max(v[0](0), std::max(v[1](0), v[2](0)))) + 1,
          y_min = std::floor(std::min(v[0](1), std::min(v[1](1), v[2](1)))) - 1,
          y_max = std::ceil(std::max(v[0](1), std::max(v[1](1), v[2](1)))) + 1;
    
    for(float pixel_x = x_min; pixel_x < x_max; pixel_x += 1)
        for(float pixel_y = y_min; pixel_y < y_max; pixel_y += 1) {
            // ansv: &#x7528;&#x4E8E;&#x5B58;&#x50A8;&#x8BE5;&#x50CF;&#x7D20;&#x70B9;&#x548C;&#x5468;&#x56F4;8&#x4E2A;&#x50CF;&#x7D20;&#x7684;&#x989C;&#x8272;&#x5747;&#x503C;
            Eigen::Vector3f ansv;
            ansv << 0, 0, 0;
            for(float x = pixel_x - 1; x <= pixel_x + 1; x += 1)
                for(float y = pixel_y - 1; y <= pixel_y + 1; y += 1) {
                    int index = get_index(x, y);
                    if(0 <= index && index < frame_buf.size())
                        ansv += frame_buf[index];
                }
            ansv /= 9;
            frame_buf[get_index(pixel_x, pixel_y)] = ansv;
        }
}

其实该代码还有一些bug，那就是应该使用一个临时的t_frame_buf来计算，避免用了修改的值。

SSAA

超分辨率采样，对于每一个像素点，分成多个采样点，分别采样最后求平均。

其实网上基本上没有一个能够很好的说明z-buffer和ssaa之间是如何操作的，所以本部分的代码的基本思路是，使用aa_depth_buf数组和aa_frame_buf数组分别记录每一个采样点的颜色以及深度，最后只是使用了aa_frame_buf来做一次平均。

void rst::rasterizer::rasterize_triangle_SSAA(const Triangle& t) {
    auto v = t.toVector4();

    float x_min = std::floor(std::min(v[0](0), std::min(v[1](0), v[2](0)))) - 1,
          x_max = std::ceil(std::max(v[0](0), std::max(v[1](0), v[2](0)))) + 1,
          y_min = std::floor(std::min(v[0](1), std::min(v[1](1), v[2](1)))) - 1,
          y_max = std::ceil(std::max(v[0](1), std::max(v[1](1), v[2](1)))) + 1;

    float nex[4] = {0.25, 0.75, 0.25, 0.75}, ney[4] = {0.25, 0.25, 0.75, 0.75};
    for(float pixel_x = x_min; pixel_x < x_max; pixel_x += 1.) {
        for(float pixel_y = y_min; pixel_y < y_max; pixel_y += 1.) {
            int index = get_index(pixel_x, pixel_y);
            Eigen::Vector3f frame_color(0, 0, 0);
            for(int i = 0; i < 4; i ++) {
                float px = pixel_x + nex[i], py = pixel_y + ney[i];
                if(!insideTriangle(px, py, v)) continue;

                auto[alpha, beta, gamma] = computeBarycentric2D(px, py, t.v);
                float w_reciprocal = 1.0/(alpha / v[0].w() + beta / v[1].w() + gamma / v[2].w());
                float z_interpolated = alpha * v[0].z() / v[0].w() + beta * v[1].z() / v[1].w() + gamma * v[2].z() / v[2].w();
                z_interpolated *= w_reciprocal;

                if(z_interpolated > aa_depth_buf[index][i]) continue;

                aa_depth_buf[index][i] = z_interpolated;
                aa_frame_buf[index][i] = t.getColor();
            }
        }
    }
}

void rst::rasterizer::SSAA_generate_frame() {
    for(int x = 0; x < width; x ++)
        for(int y = 0; y < height; y ++) {
            int index = get_index(x, y);
            frame_buf[index] = (aa_frame_buf[index][0] + aa_frame_buf[index][1]
                              + aa_frame_buf[index][2] + aa_frame_buf[index][3]) / 4.0;
        }
}

MSAA

msaa是对ssaa的优化。

对于每一个像素点来说，依旧有多个采样点，分别记录着两个信息：覆盖信息和深度信息。

对于同一个像素的每一个采样点来说，分别要进行两项测试：覆盖测试和深度测试，覆盖测试就是检查采样点是否覆盖三角形，深度测试就是检查采样点是否离摄像机最近。

在同一个像素中，假设有4个采样点，通过测试的采样点有2个，那么对像素中心取样之后乘上通过这个测试的比例（即取样后的颜色乘上四分之二）来确定该像素的颜色信息。

注意！对像素中心取样，这个就是相比与SSAA优化的地方，相比与SSAA对每一个采样点都取样，MSAA只对像素中心取样，大大优化的性能。

但是，这里有一个严重的问题：如果有多个三角形覆盖了这个像素，比如一个三角形覆盖了该像素的2个采样点，另一个三角形覆盖了该像素的另外2个采样点，那么MSAA只对像素中心进行一次采样，乘上比例之后依旧会使颜色变暗（即产生黑边）。（跑过一次，有点忘记结果了，但个人感觉大概率是这样子的）

这是因为有多个三角形覆盖，可是MSAA只进行了一次采样，缺失了其它三角形的信息，那么这里可以取一个average。

对于每一个三角形，每一个像素，对中心取样之后，把取样后的结果都存在通过测试的采样点中，最后调用SSAA_generate_frame求平均（上面SSAA代码中有）。

void rst::rasterizer::rasterize_triangle_MSAA(const Triangle& t) {
    auto v = t.toVector4();

    float x_min = std::floor(std::min(v[0](0), std::min(v[1](0), v[2](0)))) - 1,
          x_max = std::ceil(std::max(v[0](0), std::max(v[1](0), v[2](0)))) + 1,
          y_min = std::floor(std::min(v[0](1), std::min(v[1](1), v[2](1)))) - 1,
          y_max = std::ceil(std::max(v[0](1), std::max(v[1](1), v[2](1)))) + 1;
    
    float nex[4] = {0.25, 0.75, 0.25, 0.75}, ney[4] = {0.25, 0.25, 0.75, 0.75};
    for(float pixel_x = x_min; pixel_x < x_max; pixel_x += 1)
        for(float pixel_y = y_min; pixel_y < y_max; pixel_y += 1) {
            int index = get_index(pixel_x, pixel_y);
            if(!insideTriangle(pixel_x + 0.5, pixel_y + 0.5, v)) continue;
            Eigen::Vector3f t_color = t.getColor();

            for(int i = 0; i < 4; i ++) {
                float px = pixel_x + nex[i], py = pixel_y + ney[i];
                // test1
                if(!insideTriangle(px, py, v)) continue;

                auto[alpha, beta, gamma] = computeBarycentric2D(px, py, t.v);
                float w_reciprocal = 1.0/(alpha / v[0].w() + beta / v[1].w() + gamma / v[2].w());
                float z_interpolated = alpha * v[0].z() / v[0].w() + beta * v[1].z() / v[1].w() + gamma * v[2].z() / v[2].w();
                z_interpolated *= w_reciprocal;
                
                // test2
                if(z_interpolated > aa_depth_buf[index][i]) continue;

                aa_frame_buf[index][i] = t_color;
                aa_depth_buf[index][i] = z_interpolated;
            }
        }
}
